Performs PSSS correlation. Returns the index of the PSSS correlation peak in a subframe (corr_peak_pos). The value of the correlation is stored in corr_peak_value. The subframe starts at corr_peak_pos - sf_n_samples.

  • Input buffer must be subframe_size long.
