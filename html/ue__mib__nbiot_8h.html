◆ srsran_ue_mib_nbiot_t

typedef struct SRSRAN_API srsran_ue_mib_nbiot_t

This object decodes the MIB-NB from the NPBCH of an NB-IoT LTE signal.

The function srsran_ue_mib_nbiot_decode() shall be called multiple times, each passing a number of samples multiple of 19200, sampled at 1.92 MHz (that is, 10 ms of samples).

The function uses the sync_t object to find the NPSS sequence and decode the NPBCH to obtain the MIB.

The function returns 0 until the MIB is decoded.
