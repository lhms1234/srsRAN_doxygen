Transforms input uint8_t bits represented by {0, 128} to {0, 1} with AVX2 instructions, the output must have size larger than SRSRAN_AVX2_B_SIZE. Specifically, the function returns 0 if x=0 and 1 if x<0, otherwise the output is not defined.

Parameters
[in,out]xA pointer to a vector of uint8_t.
[in]lenLength of vectors x, y and z.
