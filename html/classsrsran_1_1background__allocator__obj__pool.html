Detailed Description

template<typename T, size_t BatchSize, size_t ThresholdSize>
class srsran::background_allocator_obj_pool< T, BatchSize, ThresholdSize >

Pool specialized for in allocating batches of objects in a preemptive way in a background thread to minimize latency. Note: Current implementation assumes that the pool object will outlive the background callbacks to allocate new batches

Template Parameters
Tindividual object type that is being allocated
BatchSizenumber of T objects in a batch
ThresholdSizenumber of T objects below which a new batch needs to be allocated
