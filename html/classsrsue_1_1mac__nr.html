virtual

Indicate the successful transmission of a PRACH.

Parameters
ttiThe TTI from the PHY viewpoint at which the PRACH was sent over-the-air (not to the radio).
s_idThe index of the first OFDM symbol of the specified PRACH (0 <= s_id < 14).
t_idThe index of the first slot of the specified PRACH (0 <= t_id < 80).
f_idThe index of the specified PRACH in the frequency domain (0 <= f_id < 8).
ul_carrier_idThe UL carrier used for Msg1 transmission (0 for NUL carrier, and 1 for SUL carrier).
