Detailed Description

template<class T>
class srsran::tti_semaphore< T >

Implements priority semaphore based on a FIFO queue wait . This class enqueues T type element identifiers (method push) and waits until the enqueued object is the first (method wait). The first element is released by method release. The method release_all waits for all the elements to be released.
