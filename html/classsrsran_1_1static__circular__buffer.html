Detailed Description

template<typename T, size_t N>
class srsran::static_circular_buffer< T, N >

Circular buffer with fixed, embedded buffer storage via a std::array<T, N>.

  • Single allocation at object creation for std::array. Given that the buffer size is known at compile-time, the circular iteration over the buffer may be more optimized (e.g. when N is a power of 2, % operator can be avoided)
  • not thread-safe
