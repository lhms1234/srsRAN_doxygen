Initialises or reconfigures OFDM receiver.

Note
The reconfiguration of the OFDM object considers only CP, number of PRB and optionally the FFT size
Attention
The OFDM object must be zeroed externally prior calling the initialization for first time
Parameters
qOFDM object
cfgOFDM configuration
Returns
SRSRAN_SUCCESS if the initialization/reconfiguration is successful, SRSRAN_ERROR code otherwise
