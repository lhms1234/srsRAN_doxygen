uint32_t  max_prb 
)

3GPP LTE Downlink channel estimator and equalizer. Estimates the channel in the resource elements transmitting references and interpolates for the rest of the resource grid.

The equalizer uses the channel estimates to produce an estimation of the transmitted symbol.

This object depends on the srsran_refsignal_t object for creating the LTE CSR signal.
