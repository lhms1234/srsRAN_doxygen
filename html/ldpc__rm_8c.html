Carries out the actual rate-dematching (int8_t symbols).

Parameters
[in]qA pointer to the Rate-DeMatcher (a srsran_ldpc_rm_t structure instance) that carries out the rate matching.
[in]inputThe LLRs obtained from the channel samples that correspond to the codeword to be first, rate-dematched and then decoded.
[out]outputThe rate-dematched codeword resulting from the rate-dematching operation. Shall be either initialized to all zeros or to the result of previous redundancy versions is available.
[in]ERate-matched codeword length.
[in]FNumber of filler bits.
[in]bg;Current base graph.
[in]lsCurrent lifting size.
[in]rvRedundancy version 0,1,2,3.
[in]mod_typeModulation type.
[in]NrefSize of limited buffer.
Returns
An integer: The number of useful LLR if the function executes correctly, -1 otherwise.
