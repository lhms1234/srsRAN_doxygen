Detailed Description

template<typename T>
class srsran::background_obj_pool< T >

Thread-safe object pool specialized in allocating batches of objects in a preemptive way in a background thread to minimize latency. Note: The dispatched allocation jobs may outlive the pool. To handle this, the pool state is passed to jobs via a shared ptr.
