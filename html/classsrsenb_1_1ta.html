Detailed Description

UE's FSM for controlling Time Aligment command generation.

Initially the FSM starts at idle state which transitions to Measure as soon as start is called. Measurements are collected while the FSM is in Measure state. Up to MAX_NOF_MEAS are stored. The FSM uses a minimum of MIN_NOF_MEAS measurements to compute the TA average. The TA command is triggered as soon as the TA average is higher than TA_N_THRESHOLD. After triggering a TA command, holds in prohibit state for PROHIBIT_PERIOD_MS before start collecting measurements.
