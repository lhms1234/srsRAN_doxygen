)
private

Applies the current security config on a packed NAS PDU

The PDU is ciphered and integrity protected if NAS security is configured, the header type requires security protection and the current security context is valid. The function also performs a length check.

Parameters
pduThe NAS PDU already packed inside a byte_buffer
sec_hdr_typeSecurity header type of the message
Returns
True if successful, false otherwise
