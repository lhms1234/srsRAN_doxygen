Detailed Description

It encodes a batch of example messages and compares the resulting codewords with the expected ones. Reference messages and codewords are provided in files examplesBG1.dat and examplesBG2.dat.

Synopsis: ldpc_enc_test [options]

Options:

  • -b <number> Base Graph (1 or 2. Default 1).
  • -l <number> Lifting Size (according to 5GNR standard. Default 2).
  • -R <number> Number of times tests are repeated (for computing throughput).
