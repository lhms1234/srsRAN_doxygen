Detailed Description

template<typename T, typename Tag = default_intrusive_tag>
class srsran::intrusive_double_linked_list< T, Tag >

Double Linked List of pointers of type "T" that doesn't rely on allocations. Instead, it leverages T's internal pointers to store the next and previous nodes

Template Parameters
Tnode type. Must be a subclass of intrusive_double_linked_list_element<Tag>
Tagtag of nodes. Useful to differentiate separate intrusive lists inside the same T node
