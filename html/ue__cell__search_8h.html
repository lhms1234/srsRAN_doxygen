SRSRAN_API int srsran_ue_cellsearch_scan ( srsran_ue_cellsearch_t *  q,
srsran_ue_cellsearch_result_t  found_cells[3],
uint32_t *  max_N_id_2 
)

Finds up to 3 cells, one per each N_id_2=0,1,2 and stores ID and CP in the structure pointed by found_cell. Each position in found_cell corresponds to a different N_id_2. Saves in the pointer max_N_id_2 the N_id_2 index of the cell with the highest PSR Returns the number of found cells or a negative number if error
