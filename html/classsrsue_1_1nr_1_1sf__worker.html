Detailed Description

The sf_worker class handles the PHY processing, UL and DL procedures associated with 1 subframe. It contains multiple cc_worker objects, one for each component carrier which may be executed in one or multiple threads.

A sf_worker object is executed by a thread within the thread_pool.
