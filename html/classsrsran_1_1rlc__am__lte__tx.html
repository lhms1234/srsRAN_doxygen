◆ check_sn_reached_max_retx()

void srsran::rlc_am_lte_tx::check_sn_reached_max_retx ( uint32_t  sn)
private

Helper to check if a SN has reached the max reTx threshold

Caller must hold the mutex when calling the function. If the retx has been reached for a SN the upper layers (i.e. RRC/PDCP) will be informed. The SN is not removed from the Tx window, so retransmissions of that SN can still occur.

Parameters
snThe SN of the PDU to check
