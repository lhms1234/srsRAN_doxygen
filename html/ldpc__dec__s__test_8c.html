Detailed Description

Unit test for the LDPC decoder working with 16-bit integer-valued LLRs.

It decodes a batch of example codewords and compares the resulting messages with the expected ones. Reference messages and codewords are provided in files examplesBG1.dat and examplesBG2.dat.

Synopsis: ldpc_dec_s_test [options]

Options:

  • -b <number> Base Graph (1 or 2. Default 1).
  • -l <number> Lifting Size (according to 5GNR standard. Default 2).
