◆ create_polar_decoder_ssc_c()

void* create_polar_decoder_ssc_c ( uint8_t  nMax)

Creates an SSC polar decoder structure of type pSSC, and allocates memory for the decoding buffers.

This function is exactly the same as the one for the floating-point version. Note, however, that it works with a different pSSC structure (different function pointers pSSC::f, pSSC::f, pSSC::g, pSSC::xor and pSSC::hard_bit).

Parameters
[in]nMax of the number of bits in the codeword.
Returns
A pointer to a pSSC structure if the function executes correctly, NULL otherwise.
