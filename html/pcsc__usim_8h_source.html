109 /* See ETSI GSM 11.11 and ETSI TS 102 221 for details.
110  * SIM commands:
111  * Command APDU: CLA INS P1 P2 P3 Data
112  * CLA (class of instruction): A0 for GSM, 00 for USIM
113  * INS (instruction)
114  * P1 P2 P3 (parameters, P3 = length of Data)
115  * Response APDU: Data SW1 SW2
116  * SW1 SW2 (Status words)
117  * Commands (INS P1 P2 P3):
118  * SELECT: A4 00 00 02 <file_id, 2 bytes>
119  * GET RESPONSE: C0 00 00 <len>
120  * RUN GSM ALG: 88 00 00 00 <RAND len = 10>
121  * RUN UMTS ALG: 88 00 81 <len=0x22> data: 0x10 | RAND | 0x10 | AUTN
122  * P1 = ID of alg in card
123  * P2 = ID of secret key
124  * READ BINARY: B0 <offset high> <offset low> <len>
125  * READ RECORD: B2 <record number> <mode> <len>
126  * P2 (mode) = '02' (next record), '03' (previous record),
127  * '04' (absolute mode)
128  * VERIFY CHV: 20 00 <CHV number> 08
129  * CHANGE CHV: 24 00 <CHV number> 10
130  * DISABLE CHV: 26 00 01 08
131  * ENABLE CHV: 28 00 01 08
132  * UNBLOCK CHV: 2C 00 <00=CHV1, 02=CHV2> 10
133  * SLEEP: FA 00 00 00
134  */
