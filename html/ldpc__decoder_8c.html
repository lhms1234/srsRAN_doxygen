Carries out the actual decoding with 8-bit integer-valued LLRs. It is recommended to use a 7-bit representation for the LLRs, given that all values exceeding (in magnitude) will be considered as infinity.

Parameters
[in]qA pointer to the LDPC decoder (a srsran_ldpc_decoder_t structure instance) that carries out the decoding.
[in]llrsThe LLRs obtained from the channel samples that correspond to the codeword to be decoded.
[out]messageThe message (uncoded bits) resulting from the decoding operation.
[in]cdwd_rm_lengthThe number of bits forming the codeword (after rate matching).
