◆ srsran_sync_find()

srsran_sync_find_ret_t srsran_sync_find ( srsran_sync_t *  q,
const cf_t *  input,
uint32_t  find_offset,
uint32_t *  peak_position 
)

Finds the PSS sequence previously defined by a call to srsran_sync_set_N_id_2() around the position find_offset in the buffer input.

Returns 1 if the correlation peak exceeds the threshold set by srsran_sync_set_threshold() or 0 otherwise. Returns a negative number on error (if N_id_2 has not been set)

The input signal is not modified. Any CFO correction is done in internal buffers

The maximum of the correlation peak is always stored in *peak_position
