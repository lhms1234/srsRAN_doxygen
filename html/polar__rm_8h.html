Carries out the actual rate-dematching (int8_t inputs).

Parameters
[in]qA pointer to the Rate-DeMatcher (a srsran_polar_rm_t structure instance) that carries out the rate matching.
[in]inputThe LLRs obtained from the channel samples that correspond to the codeword to be first, rate-dematched and then decoded.
[out]outputThe rate-dematched codeword resulting from the rate-dematching operation.
[in]ERate-matched codeword length.
[in]n of the codeword length.
[in]KMessage size (including CRC).
[in]ibilIndicator of bit interliaver (set to 0 to disable).
Returns
An integer: 0 if the function executes correctly, -1 otherwise.
