◆ init()

srsran::proc_outcome_t srsue::rrc::ho_proc::init ( const asn1::rrc::rrc_conn_recfg_s &  rrc_reconf)

This function implements the core of the HO procedure defined in 5.3.5.4

Right after the PHY is instructed to synchronize with the new cell, DL and UL will be suspended by the PHY until in sync again with the new cell.

Note that this function is executed by the main stack thread and needs to terminate in less than 1 ms. Any sub-task requiring more time shall use background workers.

It is important that the whole 5.3.5.4 section is executed in a single procedure step. This guarantees that the stack will not run other functions between the steps, like SR, PRACH, etc.
