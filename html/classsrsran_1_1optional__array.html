Detailed Description

template<typename T, size_t N>
class srsran::optional_array< T, N >

Array of optional items. The iteration is in order of indexes and correctly skips non-present items Pointer/References/Iterators remain valid throughout the object lifetime NOTE: The sorted iteration and pointer validation guarantees add some overhead if the array is very fragmented

Ttype of objects
Nstatic size of max nof items
