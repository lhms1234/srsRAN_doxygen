22 /******************************************************************************
23  * File: ue_sync.h
24  *
25  * Description: This object automatically manages the cell synchronization
26  * procedure.
27  *
28  * The main function is srsran_ue_sync_get_buffer(), which returns
29  * a pointer to the aligned subframe of samples (before FFT). This
30  * function should be called regularly, returning every 1 ms.
31  * It reads from the USRP, aligns the samples to the subframe and
32  * performs time/freq synch.
33  *
34  * It is also possible to read the signal from a file using the
35  * init function srsran_ue_sync_init_file(). The sampling frequency
36  * is derived from the number of PRB.
37  *
38  * The function returns 1 when the signal is correctly acquired and
39  * the returned buffer is aligned with the subframe.
40  *
41  * Reference:
42  *****************************************************************************/
