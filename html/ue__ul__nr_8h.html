SRSRAN_API int srsran_ue_ul_nr_sr_send_slot ( const srsran_pucch_nr_sr_resource_t  sr_resources[SRSRAN_PUCCH_MAX_NOF_SR_RESOURCES],
uint32_t  slot_idx,
uint32_t  sr_resource_id[SRSRAN_PUCCH_MAX_NOF_SR_RESOURCES] 
)

Decides whether the provided slot index within the radio frame is a SR transmission opportunity.

Remarks
Implemented according to TS 38.213 9.2.4 UE procedure for reporting SR
Parameters
sr_resourcesProvides the SR configuration from the upper layers
slot_idxSlot index in the radio frame
[out]sr_resource_idOptional SR resource index (or identifier)
Returns
the number of SR opportunities if the provided slot index is a SR transmission opportunity, SRSRAN_ERROR code if provided parameters are invalid
