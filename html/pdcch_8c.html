Tries to decode a DCI message from the LLRs stored in the srsran_pdcch_t structure by the function srsran_pdcch_extract_llr(). This function can be called multiple times. The location to search for is obtained from msg. The decoded message is stored in msg and the CRC remainder in msg->rnti

◆ srsran_pdcch_encode()

int srsran_pdcch_encode ( srsran_pdcch_t *  q,
srsran_dl_sf_cfg_t *  sf,
srsran_dci_msg_t *  msg,
cf_t *  sf_symbols[SRSRAN_MAX_PORTS] 
)

Encodes ONE DCI message and allocates the encoded bits to the srsran_dci_location_t indicated by the parameter location. The CRC is scrambled with the RNTI parameter. This function can be called multiple times and encoded DCI messages will be allocated to the sf_symbols buffer ready for transmission. If the same location is provided in multiple messages, the encoded bits will be overwritten.
