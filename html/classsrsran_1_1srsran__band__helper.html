◆ get_abs_freq_ssb_arfcn()

uint32_t srsran::srsran_band_helper::get_abs_freq_ssb_arfcn ( uint16_t  band,

Compute the absolute frequency of the SSB for a DL ARFCN and a band. This finds an SSB center frequency following the band SS/PBCH frequency raster provided by TS38.104 table 5.4.3.1-1 as close as possible to PointA without letting any SS/PBCH subcarrier and CORESET#0 subcarrier (if RB offset is defined) below PointA.

Parameters
scsssb subcarrier spacing.
freq_point_a_arfcnfrequency point a in arfcn notation.
coreset0_offset_rbCORESET#0 RB offset. See TS 38.213, Table 13-1,2,3
Returns
absolute frequency of the SSB in arfcn notation.
