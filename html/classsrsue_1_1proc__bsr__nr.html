Called by MAC every TTI with the current state of each LCID/LCGs.

Checks if Regular BSR must be assembled, as defined in 5.4.5. Padding BSR is assembled when explicitly called by MUX when UL DCI is received Periodic BSR is triggered by the expiration of the timers

Parameters
ttiThe current TTI
new_buffer_stateBuffer state of all LCID/LCGs at the start of the TTI
