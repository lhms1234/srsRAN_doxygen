Detailed Description

Class used to pass buffers for all channels and antennas to the radio

The class provides an abstraction to map carriers and antennas to the underlying RF channels. It provides getters and setters to access the raw pointers where the signal is stored.

It can automatically allocate and deallocate memory for the buffers if a non-zero number of subframes is passed to the constructor.
