)
inlineoverridevirtual

Informs MAC about a received PUSCH transmission for given RNTI, TTI and eNb Cell/carrier.

This function does not deallocate the uplink buffer. The function push_pdu() must be called after this to inform the MAC that the uplink buffer can be discarded or pushed to the stack

Parameters
ttithe given TTI
rntithe UE identifier in the eNb
cc_idxthe eNb Cell/Carrier identifier
nof_bytesthe number of grants carrierd by the PUSCH message
crc_resthe CRC check, set to true if the message was decoded successfully
Returns
SRSRAN_SUCCESS if no error occurs, SRSRAN_ERROR* if an error occurs
