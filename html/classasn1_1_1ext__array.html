Detailed Description

template<class T, uint32_t Nthres = ceil_frac((size_t)16, sizeof(T))>
class asn1::ext_array< T, Nthres >

This array does small buffer optimization. The array has a small stack (Nthres elements) to store elements. Once the number of elements exceeds this stack, the array allocs on the heap.

Template Parameters
T
Nthresnumber of elements T that can be stored in the stack
