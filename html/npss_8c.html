◆ srsran_npss_sync_find()

int srsran_npss_sync_find ( srsran_npss_synch_t *  q,
cf_t *  input,
float *  corr_peak_value 
)

Performs time-domain NPSS correlation. Returns the index of the NPSS correlation peak in a subframe. The frame starts at corr_peak_pos-SRSRAN_NPSS_CORR_OFFSET+frame_size/2. The value of the correlation is stored in corr_peak_value.

Input buffer must be subframe_size long.
