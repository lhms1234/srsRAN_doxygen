Detailed Description

Logical Channel Demultiplexing and MAC CE dissassemble according to TS 38.321.

Currently only SDU handling for SCH PDU processing is implemented. Downlink CE are parsed but not handled.

PDUs can be pushed by multiple HARQ processes in parallel. Handling of the PDUs is done from Stack thread which reads the enqueued PDUs from the thread-safe queue.
