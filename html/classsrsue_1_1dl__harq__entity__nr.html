Detailed Description

Downlink HARQ entity as defined in 5.3.2 of 38.321.

The class supports a configurable number of HARQ processes (up to 16).

The class is configured (init and reset) by the MAC class from the Stack thread context. Main functionality, however, is carried out from a PHY worker context.

Concurrent access from threads is protected through rwlocks.
