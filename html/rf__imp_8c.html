Opens a file-based RF abstraction.

Parameters
[out]rfDevice handle
[in]rx_filesList of pre-opened FILE* for each RX channel; NULL to disable
[in]tx_filesList of pre-opened FILE* for each TX channel; NULL to disable
[in]nof_channelsNumber of channels per direction
[in]base_srateSample rate of RX and TX files
Returns
SRSRAN_SUCCESS on success, otherwise error code
