ssb_list &  ssb_list 
)

Schedule grant for SSB.

The functions schedules the SSB according to a given periodicity. This function is a simplified version of an SSB scheduler and has several hard-coded parameters.

Parameters
[in]sl_pointSlot point carrying information about current slot.
[in]ssb_periodicityPeriodicity of SSB in ms.
[in]mibMIB message content
[out]ssb_listList of SSB messages to be sent to PHY.
Remarks
This function a is basic scheduling function that uses the following simplified assumption: 1) Subcarrier spacing: 15kHz 2) Frequency below 3GHz 3) Position in Burst is 1000, i.e., Only the first SSB of the 4 opportunities gets scheduled
