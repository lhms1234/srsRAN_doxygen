int srsran_pss_find_pss ( srsran_pss_t *  q,
const cf_t *  input,
float *  corr_peak_value 
)

Performs time-domain PSS correlation. Returns the index of the PSS correlation peak in a subframe. The frame starts at corr_peak_pos-subframe_size/2. The value of the correlation is stored in corr_peak_value.

Input buffer must be subframe_size long.
