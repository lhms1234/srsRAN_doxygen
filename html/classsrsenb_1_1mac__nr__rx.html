Detailed Description

Handles UL PDU processing.

This class implements the demuxing of UL PDUs received at the MAC layer. When the PHY decodes a valid PUSCH it passes the PDU to the MAC which in turn puts them in a thread-safe task queue to return to the calling thread as quick as possible.

The demuxing of the PDUs for all users takes place on the Stack thread which calls RLC and RRC for SDUs, or the MAC/scheduler for control elements.
