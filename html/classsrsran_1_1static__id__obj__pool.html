Detailed Description

template<typename K, typename T, size_t MAX_N>
class srsran::static_id_obj_pool< K, T, MAX_N >

Operates like a circular map, but automatically assigns the ID/key to inserted objects in a monotonically increasing way. The assigned IDs are not necessarily contiguous, as they are selected based on the available slots in the circular map

Template Parameters
Ktype of ID/key
Tobject being inserted
MAX_Nmaximum size of pool
