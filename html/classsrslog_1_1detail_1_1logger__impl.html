Detailed Description

template<typename T, typename Enum>
class srslog::detail::logger_impl< T, Enum >

The logger_impl template class contains the common functionality to loggers. Its main responsibility is to control the logging level by individually enabling or disabling the logging channels that form the logger. NOTE: Thread safe class.
