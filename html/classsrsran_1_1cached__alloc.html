Detailed Description

template<typename T>
class srsran::cached_alloc< T >

Custom Allocator that caches deallocated memory blocks in a stack to be reused in future allocations. This minimizes the number of new/delete calls, when the rate of insertions/removals match (e.g. a queue) This allocator is not thread-safe. It assumes the container is being used in a single-threaded environment, or being mutexed when altered, which is a reasonable assumption
