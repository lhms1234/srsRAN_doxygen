Detailed Description

template<typename myobj>
class srsran::multiqueue_handler< myobj >

N-to-1 Message-Passing Broker that manages the creation, destruction of input ports, and popping of messages that are pushed to these ports. Each port provides a thread-safe push(...) / try_push(...) interface to enqueue messages The class will pop from the several created ports in a round-robin fashion. The popping() interface is not safe-thread. That means, that it is expected that only one thread will be popping tasks.
