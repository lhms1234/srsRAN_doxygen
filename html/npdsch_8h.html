SRSRAN_API void srsran_npdsch_sib1_unpack ( uint8_t *const  msg,
srsran_sys_info_block_type_1_nb_t *  sib 
)

Reads the HFN from the SIB1-NB, according to TS 36.331 v13.2.0 Section 6.7.1

The function assumes that the 2 LSB read from the MIB-NB are already set inside the argument structure. It extracts the 8 MSB from the SIB1 and updates the HFN accordingly.

Parameters
msgThe packed SIB1-NB
sibThe SIB structure containing the MIB-part of the HFN
