int srsran_cqi_hl_get_L ( int  nof_prb)

Returns the number of bits to index a bandwidth part (L)

Remarks
Implemented according to TS 38.213 section 7.2.2 Periodic CSI Reporting using PUCCH, paragraph that refers to L-bit label indexed in the order of increasing frequency, where L = ceil(log2(nof_prb/k/J))
