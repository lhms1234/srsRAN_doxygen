◆ rf_uhd_imp_underflow_state_t

enum rf_uhd_imp_underflow_state_t

Transmitter finite state machine. This FSM manages the burst state of the transmitter. If the driver detects an Underflow, Overflow or Late, the FSM will make sure the burst is ended and started at the next transmission. The FSM has the following states:

  • BURST: A burst has started
  • END_OF_BURST: An underflow, overflow or late has been detected, the next transmission shall be aborted and an end of burst will be send in the next transmission;
  • WAIT_EOB_ACK: Waits for either an end of burst ACK event or a transmission after EOB_ACK_TIMEOUT_S the Late/Underflow occurred.

  • START_BURST: The next transmission will be flagged as start of burst.

    +----—+ L/O/U detected +-----------—+ EoB Sent +-----------—+ EOB ACK Rx +----------—+ | Burst |------------—>| End-of-burst |---------—>| Wait EOB ACK |---------—>| Start burst |<– Initial state +----—+ +-----------—+ +-----------—+ +----------—+ ^ | | | | New Transmission | New Transmission

    (TS timed out)
    Start of burst is transmitted

    +------------------------------------------------------—+-------------------------—+
